## 二、AI基础知识

### AI是什么？

AI（人工智能）就像一个"超级助手"：

- 它需要你给它清晰的指令
- 提示词就是给助手的"工作说明"
- 它不会取代老师，但会用提示词的老师效率会高10倍

### 推荐AI工具

| 工具名称 | 特点               | 适用场景           |
| -------- | ------------------ | ------------------ |
| Kimi     | 免费、支持长文本   | 教案、课件内容生成 |
| 文心一言 | 百度出品、中文优化 | 各类教学场景       |

### 使用方式

1. 打开AI工具网站或APP
2. 在对话框中输入提示词
3. 等待AI生成内容
4. 根据需要修改或追问

## 三、5个核心提示词技巧

### 技巧1：角色设定

**原理**：告诉AI它是谁，它会以这个身份来思考和回答

**示例**：

- ❌ "帮我写一个教案"
- ✅ "你是一位有20年经验的小学语文教师，擅长情境教学..."

**效果**：AI会以资深教师的视角来设计，更符合教学实际

### 技巧2：任务描述

**原理**：明确告诉AI你要什么，越具体越好

**关键要素**：

- 学科、年级
- 课题名称
- 课时数量
- 具体要求

**示例**：

- ❌ "写一个教案"
- ✅ "设计一节三年级语文《荷花》第一课时的教案"

### 技巧3：提供示例

**原理**：给AI看一个你喜欢的范例，它会模仿这个风格

**示例**：

```
参考风格：我希望活动像"知识抢答赛"那样，
学生分组竞争，气氛活跃
```

### 技巧4：迭代优化

**原理**：第一次生成不满意时，提出具体修改要求

**常用追问方式**：

- "请把教学目标写得更具体一些"
- "活动时间太长了，请缩短到10分钟"
- "请增加一个互动环节"

### 技巧5：结构化输出要求

**原理**：明确告诉AI你要什么格式

**示例**：

```
请按以下格式输出：
1. 活动名称
2. 活动规则
3. 所需材料
4. 教师引导语
```

## 四、提示词模板

### 模板1：教案生成

```
你是一位有20年经验的小学【学科】教师，擅长【教学特色】。

请帮我设计一节【年级】【学科】课的教案框架：
- 课题：【课题名称】
- 课时：【X】课时
- 教学目标需要包含：知识与技能、过程与方法、情感态度价值观
- 请提供：教学目标、教学重难点、教学过程（含导入、新授、练习、总结）、板书设计
```

**使用示例**：

```
你是一位有20年经验的小学语文教师，擅长情境教学。

请帮我设计一节三年级语文课的教案框架：
- 课题：《荷花》
- 课时：2课时
- 教学目标需要包含：知识与技能、过程与方法、情感态度价值观
- 请提供：教学目标、教学重难点、教学过程（含导入、新授、练习、总结）、板书设计
```

### 模板2：活动设计

```
你是一位擅长游戏化教学的小学【学科】教师。

请帮我设计一个【年级】【课题】的课堂活动：
- 活动时长：【X】分钟
- 活动目标：【具体目标】
- 学生人数：【X】人

参考风格：我希望活动像【示例描述，如"抢答游戏""角色扮演""小组竞赛"】

请提供：
1. 活动名称
2. 活动规则（学生能听懂的表述）
3. 所需材料
4. 教师引导语
```

**使用示例**：

```
你是一位擅长游戏化教学的小学数学教师。

请帮我设计一个四年级《分数的初步认识》的课堂活动：
- 活动时长：15分钟
- 活动目标：让学生理解分数的含义
- 学生人数：40人

参考风格：我希望活动像"小组竞赛"，学生分组动手操作

请提供：
1. 活动名称
2. 活动规则（学生能听懂的表述）
3. 所需材料
4. 教师引导语
```

### 模板3：课件内容

```
你是一位擅长制作教学课件的小学【学科】教师。

请帮我设计一个【年级】【课题】的PPT课件内容大纲：
- 课件页数：约【X】页
- 教学重点：【重点内容】

请按以下格式输出每一页：

【第X页】标题：______
- 核心内容（3-5个要点）
- 配图建议

要求：
1. 每页内容不超过5个要点
2. 语言适合小学生理解
3. 包含1-2个互动问题
```

**使用示例**：

```
请按以下格式输出每一页：

【第X页】标题：______
- 核心内容（3-5个要点）
- 配图建议
- 讲解提示（教师可以说的话）

要求：
```

## 五、常见问题解答

### Q1：AI生成的内容太泛泛，怎么办？

**原因**：提示词不够具体

**解决**：补充更多背景信息

- 添加学生情况描述
- 说明教学重点难点
- 提供教材版本信息

### Q2：AI生成的内容太长，怎么办？

**解决**：在提示词中添加字数限制

```
请控制在500字以内
请生成3个要点即可
```

### Q3：AI生成的内容不符合教学实际，怎么办？

**解决**：使用迭代优化

```
请把导入环节改成游戏形式
请增加学生互动的环节
请把语言改得更适合三年级学生
```

### Q4：不知道用什么AI工具？

**推荐**：

- 新手入门：豆包（界面友好）
- 教案课件：Kimi（支持长文本）
- 日常使用：文心一言、通义千问
