# AI提示词培训材料

## 概述

**培训名称**：《AI助手进课堂：小学教师提示词入门》

**目标受众**：零基础小学教师

**培训时长**：1.5-2小时

## 材料清单

| 文件                  | 说明           | 用途                                 |
| --------------------- | -------------- | ------------------------------------ |
| training-handout.docx | 培训讲义       | 学员人手一份，供培训时参考和课后复习 |
| training-slides.pptx  | 培训PPT课件    | 讲师演示用                           |
| prompt-templates.docx | 提示词模板卡片 | 可打印卡片，方便老师保存使用         |

## 使用说明

1. 培训前：打印讲义和模板卡片
2. 培训中：按PPT流程讲解，配合讲义发放
3. 培训后：模板卡片供老师日常使用
