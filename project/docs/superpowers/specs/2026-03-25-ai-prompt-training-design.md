# AI提示词培训设计文档

## 概述

**培训名称**：《AI助手进课堂：小学教师提示词入门》

**目标受众**：零基础小学教师

**培训时长**：1.5-2小时

**培训目标**：

- 理解AI是什么，消除陌生感
- 掌握5个核心提示词技巧
- 获得3个可直接使用的提示词模板
- 能够独立使用AI辅助备课

**交付物**：

- 培训PPT课件
- 培训讲义文档
- 提示词模板卡片

---

## 整体流程设计

| 环节            | 时长   | 内容                | 学习产出           |
| --------------- | ------ | ------------------- | ------------------ |
| 开场：神奇时刻  | 15分钟 | AI是什么 + 现场演示 | 激发兴趣，建立认知 |
| 案例1：生成教案 | 25分钟 | 角色设定+任务描述   | 模板1              |
| 案例2：设计活动 | 25分钟 | 提供示例+迭代优化   | 模板2              |
| 案例3：制作课件 | 25分钟 | 结构化输出要求      | 模板3              |
| 动手练习        | 20分钟 | 老师实操+答疑       | 实践经验           |
| 总结收尾        | 10分钟 | 模板发放+学习建议   | 学习路径           |

## 详细设计

### 一、开场环节：神奇时刻（15分钟）

#### 环节目标

- 消除对AI的陌生感和恐惧感
- 激发学习兴趣
- 建立正确认知：AI是工具，不是替代者

#### 具体流程

**1. 破冰提问（2分钟）**

- 问老师："你们备一节课大概需要多长时间？"
- 收集答案：1小时？2小时？更久？

**2. 神奇时刻演示（5分钟）**

- 现场打开AI工具（如Kimi/文心一言/豆包等国内可用的工具）
- 输入一个精心准备的提示词，现场生成一份完整的教案
- 让老师亲眼看到：30秒生成他们需要2小时的内容

**3. AI是什么（5分钟）**

- 用通俗比喻解释：
  - AI像一个"超级助手"，需要你给它清晰的指令
  - 提示词就是给助手的"工作说明"
  - 它不会取代老师，但会用提示词的老师效率会高10倍

**4. 培训目标预告（3分钟）**

- 今天你将学会：
  - 3个核心提示词技巧
  - 3个可直接使用的模板
  - 回去就能用AI辅助备课

#### PPT内容要点

- 封面：培训标题 + 主讲人
- 数据对比：传统备课 vs AI辅助备课的时间
- AI是什么（3页以内，图示为主）
- 今日学习目标清单

---

### 二、案例1：生成教案框架（25分钟）

#### 案例背景

- **场景**：老师需要准备一节新课的教案
- **学科示例**：三年级语文《荷花》
- **痛点**：从零开始写教案耗时，希望快速生成框架

#### 演示步骤

**1. 错误示范（3分钟）**

- 直接问："帮我写一个《荷花》的教案"
- 展示结果：泛泛而谈，缺乏针对性

**2. 正确示范（7分钟）**

- 使用提示词模板，包含角色设定和任务描述
- 展示结果：结构完整、符合小学教学特点

**3. 技巧讲解（10分钟）**

- 技巧1：**角色设定** - 告诉AI"你是一位资深小学语文教师"
- 技巧2：**任务描述** - 明确学科、年级、课文、课时

**4. 模板发放（5分钟）**

- 发放模板1，讲解如何套用

#### 提示词模板1

```
你是一位有20年经验的小学【学科】教师，擅长【教学特色】。

请帮我设计一节【年级】【学科】课的教案框架：
- 课题：【课题名称】
- 课时：【X】课时
- 教学目标需要包含：知识与技能、过程与方法、情感态度价值观
- 请提供：教学目标、教学重难点、教学过程（含导入、新授、练习、总结）、板书设计
```

### 三、案例2：设计课堂活动（25分钟）

#### 案例背景

- **场景**：老师想让课堂更有趣，需要设计互动活动
- **学科示例**：四年级数学《分数的初步认识》
- **痛点**：活动设计缺乏创意，学生参与度不高

#### 演示步骤

**1. 错误示范（3分钟）**

- 问："设计一个分数教学的活动"
- 展示结果：活动单一，缺乏可操作性

**2. 正确示范（7分钟）**

- 提供一个成功活动案例作为参考
- 展示结果：活动丰富、可操作性强

**3. 技巧讲解（10分钟）**

- 技巧3：**提供示例** - 给AI看一个你喜欢的活动范例
- 技巧4：**迭代优化** - 对结果不满意时，提出具体修改要求

**4. 模板发放（5分钟）**

#### 提示词模板2

```
你是一位擅长游戏化教学的小学【学科】教师。

请帮我设计一个【年级】【课题】的课堂活动：
- 活动时长：【X】分钟
- 活动目标：【具体目标】
- 学生人数：【X】人

参考风格：我希望活动像【示例描述，如"抢答游戏""角色扮演""小组竞赛"】

请提供：
1. 活动名称
2. 活动规则（学生能听懂的表述）
3. 所需材料
4. 教师引导语
```

---

### 四、案例3：制作课件内容（25分钟）

#### 案例背景

- **场景**：老师需要制作PPT课件，需要先整理内容
- **学科示例**：五年级科学《地球的运动》
- **痛点**：内容组织混乱，不知道每页放什么

#### 演示步骤

**1. 错误示范（3分钟）**

- 问："帮我做一个地球运动的课件"
- 展示结果：内容堆砌，没有结构

**2. 正确示范（7分钟）**

- 明确要求输出格式：每页标题+要点+配图建议
- 展示结果：结构清晰，可直接用于PPT制作

**3. 技巧讲解（10分钟）**

- 技巧5：**结构化输出要求** - 明确告诉AI你要什么格式

**4. 模板发放（5分钟）**

#### 提示词模板3

```
你是一位擅长制作教学课件的小学【学科】教师。

请帮我设计一个【年级】【课题】的PPT课件内容大纲：
- 课件页数：约【X】页
- 教学重点：【重点内容】

请按以下格式输出每一页：

【第X页】标题：______
- 核心内容（3-5个要点）
- 配图建议
- 讲解提示（教师可以说的话）

要求：
1. 每页内容不超过5个要点
2. 语言适合小学生理解
3. 包含1-2个互动问题
```

---

### 五、动手练习环节（20分钟）

#### 练习设计

- **形式**：老师用手机或电脑，现场尝试使用AI
- **任务**：选择一个自己下周要上的课题，套用模板1生成教案框架
- **支持**：讲师巡视，解答问题

#### 练习步骤

**1. 准备（3分钟）**

- 老师选择课题，打开AI工具

**2. 实操（10分钟）**

- 套用模板，生成内容

**3. 分享（5分钟）**

- 邀请2-3位老师分享成果

**4. 答疑（2分钟）**

- 解答共性问题

#### 常见问题预案

| 问题             | 解决方案                 |
| ---------------- | ------------------------ |
| AI生成的太泛     | 讲解如何补充更多背景信息 |
| AI生成的太长     | 讲解如何添加字数限制     |
| 不知道用什么工具 | 提供工具推荐清单         |

---

### 六、总结收尾环节（10分钟）

#### 环节目标

- 强化记忆
- 提供后续学习路径
- 发放学习资料

#### 具体流程

**1. 技巧回顾（3分钟）**

- 快速回顾5个技巧：
  1. 角色设定
  2. 任务描述
  3. 提供示例
  4. 迭代优化
  5. 结构化输出要求

**2. 模板发放（3分钟）**

- 发放3个提示词模板（纸质或二维码）
- 提醒：模板是起点，可以根据需要修改

**3. 后续学习建议（2分钟）**

- 多用：每天尝试用AI完成一个小任务
- 多改：对生成结果不满意时，尝试修改提示词
- 多交流：和同事分享好用的提示词

**4. 结束语（2分钟）**

- "AI不会取代老师，但会用AI的老师会更轻松"
- 感谢参与，留下联系方式供后续咨询

#### PPT内容要点

- 技巧回顾图示
- 3个模板汇总
- 推荐的AI工具清单
- 学习资源/联系方式
