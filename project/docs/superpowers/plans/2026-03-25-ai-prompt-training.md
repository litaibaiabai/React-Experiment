## 二、AI基础知识

### AI是什么？

AI（人工智能）就像一个"超级助手"：

- 它需要你给它清晰的指令
- 提示词就是给助手的"工作说明"
- 它不会取代老师，但会用提示词的老师效率会高10倍

## 三、5个核心提示词技巧

### 技巧1：角色设定

**原理**：告诉AI它是谁，它会以这个身份来思考和回答

**示例**：

- ❌ "帮我写一个教案"
- ✅ "你是一位有20年经验的小学语文教师，擅长情境教学..."

**效果**：AI会以资深教师的视角来设计，更符合教学实际

### 技巧2：任务描述

**原理**：明确告诉AI你要什么，越具体越好

**关键要素**：

- 学科、年级
- 课题名称
- 课时数量
- 具体要求

**示例**：

- ❌ "写一个教案"
- ✅ "设计一节三年级语文《荷花》第一课时的教案"

### 技巧3：提供示例

**原理**：给AI看一个你喜欢的范例，它会模仿这个风格

**示例**：
```

参考风格：我希望活动像"知识抢答赛"那样，
学生分组竞争，气氛活跃

```

### 技巧4：迭代优化

**原理**：第一次生成不满意时，提出具体修改要求

**常用追问方式**：
- "请把教学目标写得更具体一些"
- "活动时间太长了，请缩短到10分钟"
- "请增加一个互动环节"

### 技巧5：结构化输出要求

**原理**：明确告诉AI你要什么格式

**示例**：
```

请按以下格式输出：

1. 活动名称
2. 活动规则
3. 所需材料
4. 教师引导语

```

## 五、常见问题解答

### Q1：AI生成的内容太泛泛，怎么办？

**原因**：提示词不够具体

**解决**：补充更多背景信息
- 添加学生情况描述
- 说明教学重点难点
- 提供教材版本信息

### Q2：AI生成的内容太长，怎么办？

**解决**：在提示词中添加字数限制
```

请控制在500字以内
请生成3个要点即可

```

### Q3：AI生成的内容不符合教学实际，怎么办？

**解决**：使用迭代优化
```

请把导入环节改成游戏形式
请增加学生互动的环节
请把语言改得更适合三年级学生

```

### Q4：不知道用什么AI工具？

**推荐**：
- 新手入门：豆包（界面友好）
- 教案课件：Kimi（支持长文本）
- 日常使用：文心一言、通义千问

## 六、后续学习建议

### 多用
- 每天尝试用AI完成一个小任务
- 从简单任务开始，逐步增加难度

### 多改
- 对生成结果不满意时，尝试修改提示词
- 记录有效的提示词，形成自己的模板库

### 多交流
- 和同事分享好用的提示词
- 组建学习小组，互相学习
